# ふしぎ迷宮アドベンチャー

中学生向けの迷路パズルゲーム（Webブラウザ対応）

## ゲーム概要

迷路を進んで☆スターを集めながらゴールを目指すパズルゲーム。
タイムとスコアでランク判定（S/A/B）があり、ベスト記録を更新してやり込める。

- 全5ステージ＋隠しステージ
- 巡回型・追跡型の敵
- カギ＆扉ギミック
- ベスト記録はブラウザに自動保存

## 操作方法

### PC
- 矢印キー：移動
- R：リスタート
- Enter / Space：決定
- ESC：戻る

### スマホ・タブレット
- 画面下の十字キーで移動
- R ボタン：リスタート
- OK ボタン：決定

## 技術スタック

- HTML5 Canvas + Vanilla JavaScript（フレームワークなし）
- データ保存：localStorage
- ホスティング：Cloudflare Pages

## ローカルで動かす

```bash
python3 -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```
